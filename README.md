## DSTUtils

A mod for Don't Starve Together


## Installation

it needs to go in the mod folder

```powershell
C:\Program Files (x86)\Steam\steamapps\common\Don't Starve Together\mods\DSTUtils
```

You can install it by running

```powershell
New-Item -Type Folder -Path "C:\Program Files (x86)\Steam\steamapps\common\Don't Starve Together\mods\DSTUtils"
git clone https://github.com/Kronosthemad/dstutils.git "C:\Program Files (x86)\Steam\steamapps\common\Don't Starve Together\mods\DSTUtils"
```

## Dependency's

 - You Need to be on `windows` for this to work
 - You Will Need `git` to make the copy block to work
 - You will need to have `Don't Starve Together` Installed for you to get any use out of the mod
